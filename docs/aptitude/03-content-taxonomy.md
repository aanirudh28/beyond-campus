# 03 — Content Taxonomy

Hierarchy: **Domain → Topic → Subtopic → Skill → Question**.
Skills are the atomic unit of mastery tracking, ratings, and recommendations.
Questions attach to exactly one primary skill (+ optional secondary tags).

Counts below are v1 targets for the question bank (Q = approved questions).
Priority: `P0` = MVP (launch blocker), `P1` = first 90 days, `P2` = later.

## Sequencing (the default path, never a lock)

1. **Foundation loop (weeks 1–2):** Percentages → Ratios → Averages + daily
   mental-math drills. Everything else in arithmetic compounds on these.
2. **Arithmetic core (weeks 3–5):** PLD → SI/CI → TSD → Time & Work, interleaved
   with Logical P0 topics (series, coding, arrangements) and Verbal grammar/RC.
3. **Test-pattern phase (weeks 6+):** DI sets, DS, puzzles, checkpoints →
   company sims. Business domain for consulting-track students.

Onboarding placement diagnostic (05) + target-company patterns (08) reorder this
per student; the placement-date countdown compresses or relaxes it.
